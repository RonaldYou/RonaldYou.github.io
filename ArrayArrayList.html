Array/ArrayList

A more efficient method of storing data

Arrays

Arrays are used to store multiple values of a data type (primitive and composite), however, every array has a set size. Arrays are extremely useful and beneficial as now there is no need to create seperate variables per value. Below are two methods to instantiate arrays. The first method creates an array of size 3 with nothingin it while the second method creates an array of of size 3 already filled with 3 cars.


//Method 1
String car [] = new String [3];

//Method 2
String car [] = {BMW, Auidi, Toyota};

//Example 1 --> outputs BMW
System.out.println(car[1]);

//Example 2 --> outputs 3
System.out.println(car.length);

Important array functions include finding the length of an array, Example 1 by using the .length method - note: the indices of an array (the position of each value) starts from 0 making the last index of an array the length minus 1. To find the value at an index, Example 2 we use arr[i] where "arr" is the name of the array and "i" is the index shown above. When reading from an array we use the index of the array to determine which element we want to read. One way of reading from an array is using a for loop as it loops continuously until the end of the array is reached.

ArrayLists

ArrayLists are very similar to arrays. ArrayLists can also contain multiple values of a data type however arraylists have no definite size, as the amount of values increase, the size of the arraylist increases with it. The code bellow shows how to declare an arraylist which is quite different than arrays another difference is finding the length of an arraylist. Instead of using .legnth arraylists use .size(). 


//Initializing
ArrayList car = new ArrayList();

//ArrayList size --> outputs 3
System.out.println(car.size());

//Add
car.add("BMW");
car.add("Auidi");
car.add("Toyota");

//Element at index --> outputs BMW
System.out.println(car.get(1));

//Removing the first element
card.remove(1);

Arrays do not contain any methods to add or remove values however arraylists do. Once you have declared an arraylist you can add elements using a car.add("val") where "car" is the arraylist and "val" is the value being added. To remove an element in an array you would need to create a new array on the other hand, to remove from an arraylist you can use the remove function, car.remove(idx) where "car" is the arraylist and "idx" is the index of the value being removed.

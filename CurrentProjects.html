Tutorial

To create a video game with java a useful software is javafx which allows you to create GUIs (graphics user interface). Creating scenes (the screen the user interacts with) and stages (the whole gui); the scene is positioned inside the stage. In addition, you can import an image to use as your screen.


//Creates a layout
Pane root = new Pane();

//Creating a new scene
Scene scene = new Scene(root, 1200, 750);;

//Imports an image
Image img  = new Image("Background.jpg");

//Adds the image to the background
root.getChildren().add(new ImageView(img));

//Adding the scene to the stage
window.setScene(scene);

//Displaying the stage
window.show();

Another important aspect of games is buttons. Below is a creating of a button and a method to design your own buttons.


//Specialized button designs --> font, size, colour, and curvature of corners
final String IDLE_BUTTON_STYLE = "-fx-font-size:40 ;  -fx-background-color: #8e7cc3; -fx-background-radius: 15px; -fx-text-fill: #ffffff";
final String HOVERED_BUTTON_STYLE = "-fx-font-size:40; -fx-background-color: #674ea7; -fx-background-radius: 15px; -fx-text-fill: #ffffff;";
final String CLICKED_BUTTON_STYLE = "-fx-font-size:40; -fx-background-color: #674ea7; -fx-background-radius: 15px; ";

//Creating button
Button button = new Button("Click Me");

//Moving the button around
button.relocate (413,295);

//Setting button size
button.setPrefSize(375, 91);

//Default style of button
button.setStyle(IDLE_BUTTON_STYLE);

//If mouse if hovering over button
button.setOnMouseEntered(e -> start.setStyle(HOVERED_BUTTON_STYLE));

//If the mouse leaves the button
button.setOnMouseExited(e -> start.setStyle(IDLE_BUTTON_STYLE));

//If the mouse is pressed
button.setOnMousePressed(e -> start.setStyle(CLICKED_BUTTON_STYLE));

//When the mouse is clicked output "CLICK"
button.setOnAction(e -> {
	System.out.println("CLICK");
});

A part of my game that makes it unique is the animation that flips to the next scenario. The animation looks like a card flipping from the front side to the backside.


public static void flip(Group front, Group back){
	ScaleTransition stHideFront = new ScaleTransition(Duration.millis(1500), front);
        stHideFront.setFromX(1);
        stHideFront.setToX(0);

        ScaleTransition stShowBack = new ScaleTransition(Duration.millis(1500), back);
        stShowBack.setFromX(0);
        stShowBack.setToX(1);
        
        stHideFront.setOnFinished(new EventHandler() {
            @Override
            public void handle(ActionEvent t) {
                stShowBack.play();
            }
        });
        stHideFront.play();
}

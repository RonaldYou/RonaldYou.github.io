The Basics of OOP

Object-oriented programming is creating objects that contain both data and methods. Classes and objects are two main parts of OOP - in simple terms a class is a template for objects and an object is an instance of a class. A class is the blueprint of the attributes and methods of an objects. Attributes or behaviors are the distinct features of the object. Methods are the actions the object can perform or actions that can be used to manipulate the object. Below is an example of a class called cereal. The attributes are the name, the manufactuer, the calories, and the rating of the cereal. The methods of the cereal class include get methods which return the value of an attribute (getManufacturer), set methods which change the attributes of the class (changeRating). Other important aspects of a class are the contructor and toString methods. The contructor (the first method) instantiates the object while the toString method outputs a string when you try to print an object.


public class cereal{
    String name;
    char manufacturer;
    double calories, rating;

    public cereal(String n, char m, double cal, double r){
        name = n;
        manufacturer = m;
        calories = cal;
        rating = r;
    }

    public String toString(){
        return name + " cereal by " + manufacturer;
    }

    public void changeRating(double r){
        rating = r;
    }

    public double getCalories(){
        return calories;
    }

    public double getRating(){
        return rating;
    }

    public char getManufacturer(){
        return manufacturer; 
    }

}

Below is an example of a declaration and instantiation of an object of the above class - cereal. Object "c" is created by using the constructor above.

								
public class Main {
    public static void main(String[] args) {
        cereal c = new cereal("All-Bran", 'K', 70, 59.425505);
        System.out.println(c);
        System.out.println("Type: " + c.getType());
        System.out.println("Calories: " + c.getCalories());
        System.out.println("Rating: " + c.getRating());

    }
}
